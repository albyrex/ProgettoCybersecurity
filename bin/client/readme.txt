
./cert         -> folder for the certification authtority's certificate (CA_CERT.pem)

./crl          -> folder for the certification authtority's certification revocation list (CRL)

client         -> executable file for the program (1 - run with chmod +x client , 2 - ./client )

ClientCERT.pem -> certificate for the client (contains the PUB EC KEY)

priv.pem       -> EC private key for the client (for the digital signature)
