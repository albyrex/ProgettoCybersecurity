
./cert               -> folder for the certification authtority's certificate (CA_CERT.pem)

./crl                -> folder for the certification authtority's certification revocation list (CRL)

./file               -> shared folder for users file

./file_TMP           -> folder for temporary files used to upload users files to the shared folder

allowedClients.txt   -> there are all the clients allowed to talk with the server

priv.pem             -> EC private key for the server (for the digital signature)

server               -> executable file for the program (1- run with chmod +x server , 2- ./server )

ServerCERT.pem       -> certificate for the server (contains the PUB EC KEY)
